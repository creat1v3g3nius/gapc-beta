# INDEX_01_META — META (CORE)

## Objet
Index des documents **META** du CORE du framework GAPC.

Les META définissent le **socle méthodologique** :
- conventions de nommage
- règles d’écriture
- protocole de sortie
- standards de frontmatter
- boucle SOP d’exécution

## Documents

- [[META_00_HANDBOOK]] — guide général (philosophie + architecture + principes)
- [[META_01_OUTPUT_PROTOCOL]] — protocole de sortie (structure attendue des livrables)
- [[META_02_SOP_STANDARD_LOOP]] — boucle SOP standard (intake → diagnostic → plan → production → validation → next step)
- [[META_03_NAMING_CONVENTIONS]] — conventions de nommage (fichiers/IDs/dossiers)
- [[META_04_WRITING_RULES]] — règles de rédaction (H1 unique, sections courtes, lisibilité)
- [[META_05_FRONTMATTER]] — contrat YAML (champs requis + règles d’alignement)
